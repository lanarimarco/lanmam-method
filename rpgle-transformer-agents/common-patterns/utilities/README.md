# Utilities Directory

Place reusable utility code snippets here.

## Purpose
This directory contains commonly used utility functions, helper classes, and code snippets that can be reused across multiple program conversions.

## Suggested Organization

### Java Utilities
- `DateUtils.java` - Date/time conversion utilities
- `NumericUtils.java` - Packed decimal and numeric formatting
- `StringUtils.java` - String manipulation helpers specific to AS/400 conversions
- `ValidationUtils.java` - Common validation logic

### TypeScript/React Utilities
- `dateFormatters.ts` - Frontend date formatting
- `apiHelpers.ts` - Common API interaction patterns
- `formValidation.ts` - Form validation helpers

### SQL/Database
- `common-queries.sql` - Reusable SQL query patterns
- `db2-to-h2-mappings.md` - Database compatibility notes

## Best Practice
As you discover reusable patterns during conversions, extract them into utility files here for use in future conversions.
