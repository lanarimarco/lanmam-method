1. System Architecture

Overview

The rpgle-transformer-agents implements a 7-phase agent pipeline that converts RPGLE programs to modern Java/React applications with surgical precision and comprehensive documentation.

Input Layer

RPGLE Source (.rpgle) + DDS Files (.dds) + Copybooks

↓
Agent Pipeline (7 Phases)

Analysis → Database → Conversion → UI → Testing → Review → Integration

↓
Output Layer

Spring Boot Backend + React Frontend + Comprehensive Documentation

Key Design Principles

  • LLM-Agnostic: Works with any LLM (Claude, GPT-4, Gemini, local models)
  • Phase Isolation: Each phase can run independently with clear inputs/outputs
  • Functional Fidelity: Preserves exact RPGLE behavior (9.5/10 accuracy)
  • Production Ready: 80%+ test coverage, comprehensive documentation
  • Traceability: Extensive comments linking Java code to RPGLE source

4. Implementation Details

Backend Architecture Pattern

@RestController
@RequestMapping("/api/customers")
public class CustomerInquiryController {

    private final CustomerInquiryService service;

    /**
     * Get customer by number
     *
     * RPGLE Program Flow:
     * 1. User enters customer number on PROMPT screen
     * 2. Program validates and reads CUSTMAST
     * 3. If found, displays DETAIL screen
     * 4. If not found, displays error message
     */
    @GetMapping("/{customerNumber}")
    public ResponseEntity<CustomerDTO> getCustomer(
            @PathVariable Integer customerNumber) {

        log.info("REST API: Get customer request for: {}", customerNumber);
        CustomerDTO customer = service.getCustomerByNumber(customerNumber);
        return ResponseEntity.ok(customer);
    }

    // Exception handling
    @ExceptionHandler(CustomerNotFoundException.class)
    public ResponseEntity<ErrorResponse> handleNotFound(
            CustomerNotFoundException ex) {
        return ResponseEntity
            .status(HttpStatus.NOT_FOUND)
            .body(new ErrorResponse(ex.getMessage()));
    }
}

Service Layer Validation

@Service
@Transactional(readOnly = true)
public class CustomerInquiryService {

    /**
     * Validate customer number
     *
     * RPGLE validation logic:
     * If PCUSTNO = 0
     *    Eval *IN90 = *On
     *    Eval PMSG = 'Customer number required'
     */
    private void validateCustomerNumber(Integer customerNumber) {
        if (customerNumber == null || customerNumber == 0) {
            log.warn("Invalid customer number: {}", customerNumber);
            throw new CustomerValidationException(
                "Customer number required"
            );
        }

        // Additional validation (enhancement)
        if (customerNumber < 0) {
            throw new CustomerValidationException(
                "Customer number must be positive"
            );
        }
    }
}

JPA Entity with RPGLE Naming

@Entity
@Table(name = "CUSTMAST")  // Preserves RPGLE table name
@Data
@Builder
public class Customer implements Serializable {

    @Id
    @Column(name = "CUSTNO", precision = 5, scale = 0)
    private Integer customerNumber;  // Source: CUSTNO (5P 0)

    @Column(name = "CUSTNAME", length = 30)
    private String customerName;     // Source: CUSTNAME (30A)

    @Column(name = "BALANCE", precision = 9, scale = 2)
    private BigDecimal balance;      // Source: BALANCE (9P 2)

    // Note: Not used by CUST001 but included for completeness
    @Column(name = "CREDITLIM", precision = 9, scale = 2)
    private BigDecimal creditLimit;
}

Unit Test Example

@ExtendWith(MockitoExtension.class)
class CustomerInquiryServiceTest {

    @Mock
    private CustomerRepository repository;

    @InjectMocks
    private CustomerInquiryService service;

    @Test
    @DisplayName("Should throw exception when customer number is zero")
    void testZeroCustomerNumber() {
        // Given
        Integer customerNumber = 0;

        // When & Then
        assertThatThrownBy(() -> service.getCustomerByNumber(customerNumber))
            .isInstanceOf(CustomerValidationException.class)
            .hasMessage("Customer number required");

        // Verify no database call was made
        verify(repository, never()).findByCustomerNumber(any());
    }

    @Test
    @DisplayName("Should return customer when found")
    void testCustomerFound() {
        // Given
        Customer customer = Customer.builder()
            .customerNumber(12345)
            .customerName("John Doe")
            .balance(new BigDecimal("1250.75"))
            .build();

        when(repository.findByCustomerNumber(12345))
            .thenReturn(Optional.of(customer));

        // When
        CustomerDTO result = service.getCustomerByNumber(12345);

        // Then
        assertThat(result)
            .isNotNull()
            .satisfies(dto -> {
                assertThat(dto.getCustomerNumber()).isEqualTo(12345);
                assertThat(dto.getCustomerName()).isEqualTo("John Doe");
                assertThat(dto.getBalance()).isEqualByComparingTo("1250.75");
            });
    }
}

Test Categories

  • Validation Tests: Null, zero, negative numbers
  • Boundary Tests: Min (1), Max (99999)
  • Business Logic Tests: RPGLE behavior matching
  • Edge Cases: Null optional fields, BigDecimal precision
  • Integration Tests: Full stack (controller → repository)
