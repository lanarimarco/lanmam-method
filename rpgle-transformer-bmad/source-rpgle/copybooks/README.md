# Copybooks Directory

Place your RPGLE copybook files (copy members) here.

## Purpose
This directory holds reusable code snippets, data structures, and other copybook files that are included in multiple RPGLE programs using `/COPY` or `/INCLUDE` directives.

## File Naming
- Typically use `.rpgle` or `.rpgleinc` extensions
- Follow your organization's naming conventions

## Usage
During conversion, the analysis agent will identify copybook dependencies and reference these files when analyzing program structure.
